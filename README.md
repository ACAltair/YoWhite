# YoWhite
A template for Yobot

本项目仅为模板，请配合 [Yobot](https://github.com/pcrbot/yobot) 使用

This project is only a template, please use it with [Yobot](https://github.com/pcrbot/yobot)


## 如何使用 How to use

- 插件版或源码版：

直接覆盖，重启Yobot

- 便携版：

更换插件版或源码版，再进行上述操作

或自行替换文件重新打包
